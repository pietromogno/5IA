Mogno Pietro 5IA 17/10/2017

L' idea risolutiva consiste sostanzialmente nel caricare in un albero binario gli stati dell' automa
per il' "menu" sono state usate due matrici una per le pizze abbinate al costo e una per le bevande con anche "piccola" o "grande"
i programmi risultano funzionanti e rispettano tutte le richieste della consegna.
Il progetto è stato svolto insieme al compagno Trevisan Simone.
